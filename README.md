# Rover - Virtual pet

Run Rover, the virtual pet from Windows XP, on the latest version of Windows.
Watch him and interact with him!
When you click on him, he performs various actions. If you don't click on him for a long time, he may fall asleep.

![Rover](https://github.com/MalwareStudio/Rover-The-Desktop-Assistant/assets/49496834/33050f93-60f8-4e84-8530-89ad17bd5efb)

## Settings

To change the size, **`right-click`** -> **`Change size`** and select the desired size.

To close the program, **`right-click`** -> **`Close`**.
